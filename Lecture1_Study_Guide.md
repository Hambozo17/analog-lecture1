## 1️⃣ ما هي أشباه الموصلات (Semiconductors)؟

### 🔹 الشرح البسيط:
تخيل إن المواد في الطبيعة نوعين أساسيين:
- **موصلات (Conductors):** زي النحاس والألومنيوم — دي بتوصل الكهرباء بسهولة جداً
- **عوازل (Insulators):** زي البلاستيك والزجاج — دي مش بتوصل كهرباء خالص

**أشباه الموصلات** هي مواد في النص بين الاتنين! يعني ممكن توصل كهرباء وممكن لأ، حسب الظروف (الحرارة، الشوائب اللي نضيفها ليها).

### 🔹 ليه مهمة؟
- أشباه الموصلات هي **قلب كل الأجهزة الإلكترونية** اللي بنستخدمها (الموبايل، الكمبيوتر، التلفزيون).
- أهم عنصر فيها هو **السيليكون (Si)** — وده ليه السيليكون عنده 4 إلكترونات في المدار الخارجي.

### 🔹 الجدول الدوري:
من الجدول الدوري، العناصر اللي ليها من 3 لـ 5 إلكترونات تكافؤ (valence electrons) هي الأهم في أشباه الموصلات:
| المجموعة | عدد إلكترونات التكافؤ | أمثلة |
|-----------|----------------------|-------|
| III | 3 | بورون (B)، جاليوم (Ga) |
| IV | 4 | سيليكون (Si)، جيرمانيوم (Ge) |
| V | 5 | فسفور (P)، زرنيخ (As) |

<a name="2"></a>
## 2️⃣ عنصر السيليكون (Silicon)

### 🔹 الخصائص:
- السيليكون عنده **4 إلكترونات تكافؤ** (valence electrons)
- كل ذرة سيليكون بتعمل **رابطة تساهمية (covalent bond)** مع 4 ذرات جيرانها
- كل رابطة = إلكترونين مشتركين بين ذرتين

### 🔹 تأثير الحرارة:
```
🌡️ لما الحرارة تزيد → الإلكترونات في الرابطة التساهمية بتاخد طاقة كافية
→ الإلكترون بيتحرر من الرابطة → بيبقى "إلكترون حر" (free electron)
→ المكان اللي ساب فيه الإلكترون بيبقى "فجوة" (hole)
```

### 🔹 النقطة المهمة:
> **كلما زادت الحرارة ← زاد عدد الإلكترونات الحرة ← زادت التوصيلية**

---

<a name="3"></a>
## 3️⃣ أزواج الإلكترون والفجوة (Electron-Hole Pair)

### 🔹 الفكرة:
لما إلكترون يتحرر من الرابطة التساهمية، بيسيب وراه **فجوة (hole)**. الفجوة دي بتتصرف كأنها **شحنة موجبة**.

### 🔹 إيه اللي بيحصل؟
```
خطوة 1: إلكترون يتحرر من رابطة → يبقى إلكترون حر + فجوة
خطوة 2: إلكترون حر تاني ممكن ييجي يملأ الفجوة دي (Recombination)
خطوة 3: بكده الفجوة اتنقلت لمكان تاني → كأن الفجوة "اتحركت"
```

### 🔹 النتيجة المهمة:
> في شبه الموصل عندنا **نوعين من حاملات الشحنة**:
> - **إلكترونات حرة** ← شحنة سالبة (-) 
> - **فجوات** ← بتتصرف كشحنة موجبة (+)
> 
> **الاتنين بيشاركوا في توصيل التيار!**

---

<a name="4"></a>
## 4️⃣ كثافة الإلكترونات الحرة وطاقة الفجوة (Bandgap Energy)

### 🔹 إيه هي طاقة الفجوة (Eg)؟
هي **كمية الطاقة اللي الإلكترون محتاجها عشان يتحرر من الرابطة التساهمية** ويبقى حر.

### 🔹 العلاقة الأساسية:
```
ni = B × T^(3/2) × e^(-Eg / 2kT)
```

حيث:
- **ni** = كثافة الإلكترونات الحرة الأصلية (intrinsic carrier concentration)
- **B** = ثابت يعتمد على نوع المادة
- **T** = درجة الحرارة بالكلفن
- **Eg** = طاقة الفجوة (bandgap energy)
- **k** = ثابت بولتزمان
- **e** = الأس (exponential)

### 🔹 الملخص:
> **العلاقة أُسّية (exponential):**
> - طاقة فجوة أكبر ← إلكترونات حرة أقل (أصعب تتحرر)
> - حرارة أعلى ← إلكترونات حرة أكتر

| المادة | طاقة الفجوة Eg (eV) |
|--------|---------------------|
| Si | 1.12 |
| Ge | 0.66 |
| GaAs | 1.42 |

<a name="5"></a>
## 5️⃣ التطعيم (Doping) ⭐ مهم جداً

### 🔹 إيه هو التطعيم؟
هو **إضافة شوائب (عناصر تانية) لشبه الموصل النقي** عشان نغير خصائصه الكهربائية.

---

### 🔸 النوع الأول: N-type (سالب)

**الطريقة:** نضيف عنصر من المجموعة V (زي الفسفور P) — اللي عنده **5 إلكترونات تكافؤ**

**إيه اللي بيحصل:**
```
ذرة فسفور (5 إلكترونات) تدخل مكان ذرة سيليكون (4 إلكترونات)
→ 4 إلكترونات بتعمل روابط مع الجيران
→ الإلكترون الخامس بيبقى حر! 🆓
→ كده عدد الإلكترونات الحرة زاد
→ المادة بقت N-type (الإلكترونات هي حاملات الشحنة الأساسية)
```

**المصطلحات:**
- العنصر المضاف (الفسفور) اسمه **Donor** (مانح) — لأنه بيتبرع بإلكترون
- **حاملات الشحنة الأساسية (Majority Carriers):** إلكترونات
- **حاملات الشحنة الأقلية (Minority Carriers):** فجوات

---

### 🔸 النوع التاني: P-type (موجب)

**الطريقة:** نضيف عنصر من المجموعة III (زي البورون B) — اللي عنده **3 إلكترونات تكافؤ**

**إيه اللي بيحصل:**
```
ذرة بورون (3 إلكترونات) تدخل مكان ذرة سيليكون
→ 3 إلكترونات بس بتعمل روابط
→ الرابطة الرابعة ناقصها إلكترون → يبقى عندنا فجوة! 🕳️
→ كده عدد الفجوات زاد
→ المادة بقت P-type (الفجوات هي حاملات الشحنة الأساسية)
```

**المصطلحات:**
- العنصر المضاف (البورون) اسمه **Acceptor** (مستقبل) — لأنه بيستقبل إلكترون
- **حاملات الشحنة الأساسية (Majority Carriers):** فجوات
- **حاملات الشحنة الأقلية (Minority Carriers):** إلكترونات

---

### 📊 جدول مقارنة مهم جداً:

| | N-type | P-type |
|---|--------|--------|
| **العنصر المضاف** | من المجموعة V (فسفور P) | من المجموعة III (بورون B) |
| **عدد إلكترونات التكافؤ** | 5 | 3 |
| **اسم الشائبة** | Donor (مانح) | Acceptor (مستقبل) |
| **Majority Carriers** | إلكترونات | فجوات |
| **Minority Carriers** | فجوات | إلكترونات |

<a name="6"></a>
## 6️⃣ كثافة الإلكترونات والفجوات

### 🔹 القانون الأساسي (مهم جداً!):

```
n × p = ni²
```

- **n** = كثافة الإلكترونات
- **p** = كثافة الفجوات
- **ni** = كثافة حاملات الشحنة الأصلية (intrinsic concentration)

### 🔹 الشرح:
> **حاصل ضرب كثافة الإلكترونات × كثافة الفجوات = ثابت دايماً = ni²**
> 
> مهما كان التطعيم! يعني لو زودنا الإلكترونات، الفجوات هتقل تلقائياً، والعكس.

### 🔹 للـ N-type:
```
n ≈ ND  (تقريباً = كثافة المانحات)
p = ni² / ND  (الفجوات بتقل)
```

### 🔹 للـ P-type:
```
p ≈ NA  (تقريباً = كثافة المستقبلات)
n = ni² / NA  (الإلكترونات بتقل)
```

<a name="7"></a>
## 7️⃣ آلية نقل الشحنة الأولى: الانجراف (Drift)

### 🔹 التعريف:
> **الانجراف (Drift)** هو حركة حاملات الشحنة بسبب وجود **مجال كهربائي (Electric Field)**

### 🔹 الفكرة:
```
لما نطبق جهد على شبه الموصل
→ بيتكوّن مجال كهربائي (E)
→ الإلكترونات بتتحرك عكس اتجاه المجال
→ الفجوات بتتحرك في اتجاه المجال
→ الاتنين بينتجوا تيار!
```

### 🔹 السرعة:
```
v = μ × E
```
- **v** = سرعة الانجراف (drift velocity)
- **μ** = الحركية (mobility) — مدى سهولة حركة الشحنة في المادة
- **E** = المجال الكهربائي

### 🔹 تيار الانجراف:
```
J_drift = q × (n × μn + p × μp) × E
```
- **q** = شحنة الإلكترون (1.6 × 10⁻¹⁹ C)
- **n, p** = كثافة الإلكترونات والفجوات
- **μn, μp** = حركية الإلكترونات والفجوات

### 🔹 نقطة مهمة:
> تيار الانجراف **يعتمد على المجال الكهربائي** — كلما كان المجال أقوى، كان التيار أكبر.

---

<a name="8"></a>
## 8️⃣ آلية نقل الشحنة الثانية: الانتشار (Diffusion)

### 🔹 التعريف:
> **الانتشار (Diffusion)** هو حركة حاملات الشحنة من **منطقة تركيز عالي** إلى **منطقة تركيز منخفض**

### 🔹 مثال من الحياة اليومية:
```
تخيل إنك حطيت نقطة حبر في كوب مية 🎨💧
← الحبر بينتشر من المنطقة المركزة لبرّه لحد ما يتوزع بالتساوي
← نفس الكلام بيحصل مع الإلكترونات والفجوات!
```

### 🔹 تيار الانتشار:
```
J_diff = q × Dn × (dn/dx) - q × Dp × (dp/dx)
```
- **Dn, Dp** = معامل الانتشار للإلكترونات والفجوات
- **dn/dx** = تدرج تركيز الإلكترونات (concentration gradient)
- **dp/dx** = تدرج تركيز الفجوات

### 🔹 ملاحظة مهمة:
> تيار الانتشار **يعتمد على تدرج التركيز** (msh المجال الكهربائي!)

### 🔹 مقارنة بين تيار الانجراف وتيار الانتشار:

| | الانجراف (Drift) | الانتشار (Diffusion) |
|---|---|---|
| **السبب** | مجال كهربائي (E) | تدرج التركيز (dn/dx) |
| **الاتجاه** | في اتجاه المجال (للفجوات) | من تركيز عالي لتركيز منخفض |
| **المعادلة** | J = qnμE | J = qD(dn/dx) |

### 🔹 تدرج التركيز الخطي vs غير الخطي:
- **خطي (Linear):** التركيز بيقل بشكل منتظم → تيار الانتشار **ثابت**
- **غير خطي (Nonlinear):** التركيز بيقل بشكل غير منتظم → تيار الانتشار **متغير**

<a name="9"></a>
## 9️⃣ علاقة أينشتاين (Einstein's Relation)

### 🔹 العلاقة:
```
D/μ = kT/q = VT
```
- **D** = معامل الانتشار
- **μ** = الحركية
- **VT** = الجهد الحراري (Thermal Voltage) ≈ **26 mV** عند درجة حرارة الغرفة (T = 300K)

### 🔹 الأهمية:
> العلاقة دي بتربط بين الانتشار والانجراف — يعني لو عرفت الحركية (μ)، تقدر تحسب معامل الانتشار (D) والعكس!

<a name="10"></a>
## 🔟 وصلة PN (الدايود) ⭐⭐ الجزء الأهم!

### 🔹 إيه هو الدايود؟
> لما **نحط منطقة N-type جنب منطقة P-type** في نفس قطعة شبه الموصل، بيتكوّن عندنا **وصلة PN** وهي الدايود!

```
┌──────────┬──────────┐
│  P-type  │  N-type  │
│ (فجوات  │(إلكترونات│
│  كتير)   │  كتير)   │
└──────────┴──────────┘
```

### 🔹 إيه اللي بيحصل عند الوصلة؟

**خطوة 1: الانتشار**
```
الإلكترونات الكتير في N → بتنتشر ناحية P
الفجوات الكتير في P → بتنتشر ناحية N
→ ده بينتج تيار انتشار (Diffusion Current)
```

**خطوة 2: تكوين منطقة النضوب (Depletion Region)**
```
الإلكترونات اللي سابت N → سابت وراها أيونات موجبة ثابتة (+)
الفجوات اللي سابت P → سابت وراها أيونات سالبة ثابتة (-)
→ المنطقة دي خالية من حاملات الشحنة الحرة
→ اسمها "منطقة النضوب" (Depletion Region)
```

**خطوة 3: تكوين مجال كهربائي**
```
الأيونات الثابتة بتكوّن مجال كهربائي ⚡
→ المجال اتجاهه من N إلى P
→ المجال ده بيعمل تيار انجراف (Drift Current) عكس تيار الانتشار
```

**خطوة 4: حالة الاتزان (Equilibrium)**
```
بعد شوية → تيار الانجراف = تيار الانتشار (بس في اتجاه معاكس)
→ صافي التيار = صفر! ✅
→ ده حالة الاتزان
```

### 🔹 الجهد المدمج (Built-in Potential) V₀:
```
V₀ = (kT/q) × ln(NA × ND / ni²)
```
- ده الجهد اللي بيتكوّن عبر الوصلة تلقائياً في حالة الاتزان
- **عند درجة حرارة الغرفة للسيليكون ≈ 0.7 V تقريباً**

<a name="12"></a>
## 1️⃣2️⃣ الانحياز العكسي (Reverse Bias)

### 🔹 كيف يحصل؟
```
نوصّل الجهد العالي على N والجهد المنخفض على P
أي: V_N > V_P
```

### 🔹 إيه اللي بيحصل؟
```
→ منطقة النضوب بتتوسع 📏
→ المجال الكهربائي المدمج بيزيد ⚡⚡
→ بيمنع أي تيار انتشار تقريباً
→ التيار العكسي صغير جداً جداً (≈ بتاع الانجراف بس)
→ عملياً: الدايود مش بيوصّل! ❌
```

### 🔹 تطبيق مهم: مكثف متغير بالجهد (Voltage-Dependent Capacitor)
```
الوصلة PN في الانحياز العكسي بتتصرف كمكثف (Capacitor)!

       ┌─── P ───┤   الفراغ   ├─── N ───┐
       └─────────┤(Depletion) ├─────────┘
                 │  ← العرض → │

لما نزوّد الجهد العكسي VR:
→ منطقة النضوب بتتوسع
→ السعة (Capacitance) بتقل
→ يعني: C يعتمد على VR!
```

### 🔹 معادلة السعة:
```
Cj = Cj0 / (1 + VR/V0)^m
```
- **Cj0** = السعة عند صفر انحياز
- **VR** = الجهد العكسي المطبق
- **V0** = الجهد المدمج
- **m** = ثابت يعتمد على شكل التطعيم (عادة ½)

### 🔹 تطبيق: المذبذب المتحكم بالجهد (VCO - Voltage Controlled Oscillator)
```
بنستخدم وصلة PN كمكثف + ملف (L) في دائرة رنين
→ تردد الرنين f = 1/(2π√(LC))
→ لما نغير VR → C بتتغير → التردد بيتغير!
→ كده تحكمنا في التردد باستخدام الجهد! 🎛️
```

<a name="13"></a>
## 1️⃣3️⃣ الانحياز الأمامي (Forward Bias) ⭐

### 🔹 كيف يحصل؟
```
نوصّل الجهد العالي على P والجهد المنخفض على N
أي: V_P > V_N
```

### 🔹 إيه اللي بيحصل؟
```
→ منطقة النضوب بتتقلص 📏⬇️
→ المجال الكهربائي المدمج بيقل ⚡⬇️
→ تيار الانتشار يقدر يتغلب على حاجز المجال
→ الإلكترونات والفجوات بتعدّي الوصلة
→ التيار بيمشي! ✅
```

### 🔹 تيار الانتشار في الانحياز الأمامي:
```
لما الانحياز الأمامي يزيد:
→ حاملات الشحنة الأقلية (minority carriers) بتزيد عند الوصلة
→ تيار الانتشار بيزيد بشكل أسّي (exponential)
```

### 🔹 تدرج حاملات الشحنة الأقلية (Minority Charge Gradient):
```
لما الأقلية (مثلاً: إلكترونات في P) تعدّي الوصلة:
→ بتتجمع عند حافة منطقة النضوب
→ كل ما تتعمق جوّه في المنطقة P
→ بيحصل إعادة اتحاد (Recombination) مع الفجوات
→ عددها بيقل تدريجياً
→ ده بينتج تدرج التركيز اللي بيسبب تيار الانتشار
```

### 🔹 ملخص الانحياز الأمامي:
```
عند الوصلة: تيار الانتشار (Minority carriers) هو المسيطر
بعيد عن الوصلة: تيار إعادة الاتحاد (Majority carriers) هو المسيطر
مجموع التيارين = ثابت (Total current ثابت في كل مكان)
```

<a name="14"></a>
## 1️⃣4️⃣ منحنى الخصائص IV للدايود

### 🔹 معادلة الدايود (The Diode Equation):
```
I = Is × (e^(VD/VT) - 1)
```
- **I** = التيار المار في الدايود
- **Is** = تيار التشبع العكسي (صغير جداً ~10⁻¹⁵ A)
- **VD** = الجهد المطبق على الدايود
- **VT** = الجهد الحراري ≈ 26 mV @ 300K

### 🔹 شكل المنحنى:
```
 التيار (I)
    ↑
    │         ╱  ← انحياز أمامي (تيار يزيد بشكل أسّي)
    │        ╱
    │       ╱
    │      ╱
    │     ╱
    │    ╱
────┤───╱──────────────── الجهد (V)
    │  ╱
    │ (≈0.7V للسيليكون)
    │
 ←──┤── انحياز عكسي (تيار صغير جداً ≈ -Is)
    │
```

### 🔹 ملاحظات:
- في **الانحياز الأمامي**: التيار يزيد بشكل أسّي بعد ≈ 0.7V
- في **الانحياز العكسي**: التيار ثابت تقريباً وصغير جداً = Is

### 🔹 دايودات متوازية:
لو وصلنا دايود تاني متوازي → التيار الكلي = مجموع التيارين
لو الدايودين متساويين → التيار ينقسم بالتساوي

## 1️⃣5️⃣ نموذج الجهد الثابت للدايود (Constant-Voltage Model)

### 🔹 الفكرة:
بدل ما نستخدم المعادلة الأسّية المعقدة، نبسّط الدايود كالتالي:

```
┌─────────────────────────────────────┐
│        نموذج الجهد الثابت          │
│                                     │
│  لو VD < VD,on (≈ 0.7V):          │
│  → الدايود مش شغال (دائرة مفتوحة) │
│  → التيار = صفر                    │
│                                     │
│  لو VD ≥ VD,on (≈ 0.7V):          │
│  → الدايود شغال                    │
│  → الجهد عليه = 0.7V (ثابت)       │
│  → التيار يتحدد من الدائرة الخارجية│
└─────────────────────────────────────┘
```

### 🔹 ليه النموذج ده أسهل؟
```
المعادلة الأسّية: I = Is × (e^(VD/VT) - 1)
→ محتاج حل تكراري (iterative) → صعب! 😰

نموذج الجهد الثابت: VD = 0.7V (لو شغال)
→ محتاج معادلات خطية بس → سهل! 😊
```

### 🔹 مثال:
```
لو عندنا دائرة: بطارية 5V + مقاومة 1kΩ + دايود

باستخدام نموذج الجهد الثابت:
V_R = 5V - 0.7V = 4.3V
I = V_R / R = 4.3V / 1000Ω = 4.3 mA

سهل وبسيط! ✅
```

## 📝 ملخص نهائي للحفظ السريع

### أهم النقاط:
| الموضوع | النقطة الأساسية |
|---------|----------------|
| **السيليكون** | 4 إلكترونات تكافؤ، روابط تساهمية |
| **N-type** | +فسفور (5 إلكترونات) = إلكترونات حرة زيادة |
| **P-type** | +بورون (3 إلكترونات) = فجوات زيادة |
| **n×p = ni²** | حاصل الضرب ثابت دايماً! |
| **الانجراف** | حركة بسبب مجال كهربائي |
| **الانتشار** | حركة من تركيز عالي لمنخفض |
| **علاقة أينشتاين** | D/μ = VT ≈ 26mV |
| **منطقة النضوب** | بتتوسع في العكسي، بتتقلص في الأمامي |
| **V₀** | الجهد المدمج ≈ 0.7V للسيليكون |
| **معادلة الدايود** | I = Is(e^(V/VT) - 1) |
| **نموذج الجهد الثابت** | VD = 0.7V لو شغال، 0 لو مش شغال |

---

### 🔑 المعادلات الأساسية للحفظ:
```
1. n × p = ni²
2. J_drift = q(nμn + pμp)E
3. J_diff = qDn(dn/dx) - qDp(dp/dx)
4. D/μ = kT/q = VT ≈ 26mV
5. V₀ = VT × ln(NA×ND/ni²)
6. I = Is × (e^(VD/VT) - 1)
7. Cj = Cj0 / (1 + VR/V₀)^m
```
