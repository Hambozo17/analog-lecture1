1. ما هي أشباه الموصلات؟

▼

الفكرة الأساسية

موصلات (نحاس — بتوصل كهرباء) | عوازل (بلاستيك — مش بتوصل) | أشباه الموصلات في النص — توصيلها يتغير حسب الحرارة أو الشوائب.

💡 أشباه الموصلات = قلب كل الإلكترونيات. وصلة PN هي أبسط وأهم عنصر.

العناصر المهمة

المجموعةإلكترونات تكافؤمثالالدور
III3بورون (B)Acceptor → P-type
IV4سيليكون (Si)شبه الموصل الأساسي
V5فسفور (P)Donor → N-type

Core Concept

Conductors (copper — conduct electricity) | Insulators (plastic — don't conduct) | Semiconductors are in between — their conductivity changes with temperature or added impurities.

💡 Semiconductors = the heart of all electronics. The PN junction is the most fundamental device.

Important Elements

GroupValence e⁻ExampleRole
III3Boron (B)Acceptor → P-type
IV4Silicon (Si)Primary semiconductor
V5Phosphorus (P)Donor → N-type

2. السيليكون

▼

الخصائص

  • 4 إلكترونات تكافؤ — 4 روابط تساهمية مع الجيران
  • كل رابطة = إلكترونين مشتركين

📊 الروابط التساهمية

كل دائرة زرقاء = إلكترون مشترك

تأثير الحرارة 🌡️

حرارة أعلى ← إلكترونات تتحرر ← إلكترون حر + فجوة

🔑 حرارة ↑ ← إلكترونات حرة ↑ ← توصيلية ↑

Properties

  • 4 valence electrons — forms 4 covalent bonds with neighbors
  • Each bond = 2 shared electrons

📊 Covalent Bonds

Each blue circle = shared electron in covalent bond

Temperature Effect 🌡️

Higher temperature → electrons gain energy → break free → free electron + hole

🔑 Temperature ↑ → free electrons ↑ → conductivity ↑

3. إلكترون وفجوة

▼

الآلية

  1. Generation: إلكترون يتحرر ← إلكترون حر + فجوة
  2. Recombination: إلكترون يملأ فجوة
  3. الفجوة تتحرك كأنها شحنة موجبة
📌 نوعين من حاملات الشحنة: إلكترونات (−) و فجوات (+)

❓ الفرق بين Generation و Recombination؟

(اضغط للإجابة)

Generation: إلكترون يتحرر → e⁻ + h⁺
Recombination: e⁻ يملأ h⁺ → الاتنين يختفوا

Mechanism

  1. Generation: Electron breaks free → free electron + hole
  2. Recombination: Free electron fills a hole
  3. Holes behave as positive charges
📌 Two types of charge carriers: Electrons (−) and Holes (+)

❓ Difference between Generation & Recombination?

(click to flip)

Generation: Electron breaks free → e⁻ + h⁺
Recombination: e⁻ fills h⁺ → both disappear

10. وصلة PN (الدايود)

▼

📊 وصلة PN

الخطوات

  1. انتشار: e⁻ من N → P، h⁺ من P → N
  2. منطقة النضوب: أيونات ثابتة ← منطقة خالية
  3. مجال كهربائي: من N لـ P ← تيار انجراف
  4. اتزان: drift = diffusion ← I = 0

الجهد المدمج

🔋 Si @ room temp: V₀ ≈ 0.7V

📊 PN Junction

Steps at the Junction

  1. Diffusion: e⁻ from N → P, h⁺ from P → N
  2. Depletion Region: Fixed ions → carrier-free zone
  3. Electric Field: From N to P → drift current
  4. Equilibrium: drift = diffusion → I = 0

Built-in Potential

🔋 Si @ room temp: V₀ ≈ 0.7V

12. الانحياز العكسي

▼

📊 الانحياز العكسي

V_N > V_P ← منطقة النضوب بتتوسع ← I ≈ 0

مكثف متغير

VR ↑ → العرض ↑ → السعة ↓

🎛️ VCO

نغير VR → C تتغير → التردد f يتغير!

📊 Reverse Bias

V_N > V_P → depletion region widens → I ≈ 0

Voltage-Dependent Capacitor

VR ↑ → width ↑ → capacitance ↓

🎛️ VCO Application

Change VR → C changes → frequency f changes!

✅

13. الانحياز الأمامي

▼

📊 الانحياز الأمامي

V_P > V_N ← النضوب بتتقلص ← التيار يمشي!

الملخص

عند الوصلة: Diffusion (minority) مسيطر
بعيد: Recombination (majority) مسيطر
المجموع = ثابت!

📊 Forward Bias

V_P > V_N → depletion shrinks → current flows!

Summary

At junction: Diffusion current (minority carriers) dominates
Far from junction: Recombination current (majority) dominates
Total = constant everywhere!
